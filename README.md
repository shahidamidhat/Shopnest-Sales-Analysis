
# 🛒 Shopnest E-commerce Sales Analysis

## 📌 Project Description
This project focuses on analyzing an e-commerce retail dataset to uncover insights related to **sales performance, customer behavior, and order delivery efficiency**.

The dataset was cleaned, transformed, and analyzed using data analytics techniques to identify key trends in **revenue generation, product categories, and regional performance**. The project also evaluates operational aspects such as **on-time vs delayed deliveries**, helping to highlight areas for business improvement.

Feature engineering techniques were applied to derive meaningful variables such as **monthly trends, delivery status, and customer behavior metrics**, enabling deeper analysis.

---

## 🎯 Key Objectives
- Analyze overall sales and revenue trends  
- Identify top-performing product categories and regions  
- Evaluate customer purchasing behavior and product ratings  
- Analyze order delivery performance (on-time vs delayed)  
- Identify seasonal demand patterns  

---

## 🛠️ Tools & Technologies 
- **Power BI** – Dashboard creation and visualization  
- **Excel** – Data preprocessing and validation  

---

## 📊 Key Analysis Performed
- Data cleaning and preprocessing of multi-table datasets  
- Feature engineering (month extraction, delivery status, revenue metrics)  
- Sales trend analysis (monthly and category-wise)  
- Regional sales distribution analysis  
- Delivery performance analysis (delayed vs on-time orders)  
- Customer behavior analysis using payment methods and product ratings  

---

## 📈 Key Insights
- Identified **top-selling product categories contributing ~35% of total revenue**  
- Discovered **seasonal sales patterns and peak demand periods**  
- Highlighted regions with **highest and lowest sales performance**  
- Analyzed delivery delays impacting customer experience  
- Derived insights on **customer preferences and purchasing behavior**  

---

## 🚀 Outcome
The analysis provided actionable insights to improve **inventory planning, marketing strategies, and operational efficiency**, enabling more data-driven business decisions.
